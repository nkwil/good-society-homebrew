# Good Society (Homebrew)

A standalone Foundry VTT v13 system implementing a homebrew variant of the Good Society TTRPG (a Jane Austen-inspired narrative game by Storybrewers Roleplaying). It replicates the printed character sheet, automates the token economy (resolve tokens, MT, monologues, cycle phases), adds a Public Info / Facilitator dashboard, interactive scene maps with NPC hover tooltips, and a Magic/Skills system with on-screen casting VFX. Forked from [Asacolips' Boilerplate System](https://github.com/asacolips-projects/boilerplate).
